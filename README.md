# 马儿蹦跳模型仓库
## 介绍
欢迎来到马儿蹦跳Mod的模型仓库！  
这里存放着马儿蹦跳Mod的官方模型包以及由马儿蹦跳开发组推荐的模型教程与建模规范。  

## 模型包
官方模型包存放着所有在马儿蹦跳Mod当中使用（或即将使用）的模型。  
如果你打算为马儿蹦跳Mod添加任何你希望在Mod中出现的模型，请提交Pull Request，并确认你的模型无任何争议或侵权内容。
提交Pull Request代表你允许本Mod将模型以CC BY-NC-SA 4.0协议使用。

## 内容
官方模型包  
模型制作教程  
推荐建模规范  
骨骼表
